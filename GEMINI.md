# Gemini Instructions

This repository is a portable paper-revision skill. Read [`SKILL.md`](SKILL.md) and follow it for academic manuscript revision work.

Use the workflow for:

- LaTeX manuscript revision
- Supplement revision
- Reviewer response letters
- Proof corrections
- Redline generation
- PDF-backed response-location audits
- Submission package checks

Expected behavior:

- Make a bounded plan under `references/` before substantive edits.
- Preserve baselines and stable current filenames.
- Apply only the approved plan.
- Build the relevant PDFs and redlines when possible.
- Verify claims from the compiled PDF or recorded evidence.
- Keep unresolved issues visible.

Do not treat this repository as a paper project. It is the reusable workflow package.
